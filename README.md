# Ranktify Backend

## Installation

1. **Install Go**: Download and install Go by following the instructions at [https://go.dev/doc/install](https://go.dev/doc/install).

2. **Install Dependencies**:
   - Download dependencies:
     ```sh
     go mod tidy
     ```
